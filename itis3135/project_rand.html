Project Design

Project Overview

  1. Describe your application and its purpose: My application is a website that allows the user to view and select different landscaping options and designs. The user has many options such as, but not limited to, being able to book a meeting online, locating the address using an interactive google map, and selecting pre-made design plans.
  2. Intended users of the website: Intended users of the website are those looking to improve their property through various landscaping opportunities. Anyone that owns property.
  3. Overview of the content of the website: The website will include a home page that describes the goal of the website, lists the different services, and provides links to other aspects of the website. The links on the home page will send the user to each of the different landscaping services offered as well as a way to contact us or book a meeting. There will also be a gallery link that shows different examples of the services provided.
